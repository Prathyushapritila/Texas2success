## ğŸ¨ Customization

### Content Management

All content is centralized in `/data/siteData.ts`. Edit this file to update:
- Company information
- Services descriptions
- Testimonials
- Case studies
- Team members
- Contact information

### Colors & Styling

- **Primary Colors**: Edit `tailwind.config.js` to customize the color scheme
- **Dark Mode**: Automatically handled via `next-themes`
- **Glassmorphism**: Defined in `app/globals.css` (`.glass` class)

### Animations

- Framer Motion animations are configured in each component
- Adjust animation delays and durations in component files
- Scroll-based animations use `whileInView` prop

## ğŸ› Troubleshooting

### Dark Mode Not Working
- Ensure `ThemeProvider` wraps your app in `layout.tsx`
- Check browser console for errors

### Animations Not Showing
- Verify Framer Motion is installed: `npm list framer-motion`
- Check viewport settings in animation props

### Build Errors
- Clear `.next` folder: `rm -rf .next`
- Reinstall dependencies: `rm -rf node_modules && npm install`
