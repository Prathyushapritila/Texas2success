# Service Images

Add service images here for each service:

- `erp-crm.jpg` - Image for Custom ERP & CRM Software service
- `salesforce.jpg` - Image for Salesforce Development service

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Size**: 1200x800px recommended
- **Style**: Professional, modern, related to the service
- **Optimization**: Compress images before adding
